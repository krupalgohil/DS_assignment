# Insurance Attach Percentage Analysis

This project analyzes insurance attach percentage data for Jumbo & Company (Zopper) as part of a Data Science Internship assignment.

## Objective
- Analyze attach performance across branches, months, and stores  
- Identify performance variations and improvement opportunities  
- Categorize stores based on attach performance  
- Predict January attach percentage using recent trends  

## Tools Used
- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook  

## Dataset
- Store-level insurance attach percentage data  
- Time period: August to December  

## Key Analysis
- Branch-wise analysis  
- Month-wise trend analysis  
- Store-level performance analysis  
- Store categorization (High, Medium, Low)  
- January attach percentage prediction  

## Output
- Visualizations stored in the `visuals/` folder  
- Final report prepared using LaTeX
